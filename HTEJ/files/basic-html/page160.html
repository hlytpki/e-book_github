蘭嶼之星－風雨交加中自行出航的梅利號

                                                                             書記官 謝佳宏




                          記得 102 年 9 月份知道它在無                           本分署為利於拍賣程序之進行，

                     人駕駛的情況下自己出海的事蹟以                              每一次拍賣皆借用海巡署富岡漁港安
                     後，心中就一直有個念頭－我要去看                             檢所辦公室進行船舶拍賣程序。惟蘭
                     看它，見證現實生活中竟真有這麼一                             嶼之星歷經 3 次拍賣，皆未拍定，

                     艘跟梅利號 ( 日本航海王動漫 ) 一樣                         行政執行官為求能將船舶順利拍賣換
                     會自己出航的船舶―蘭嶼之星。                               價，數次督同書記官親臨富岡漁港現
                                                                  場，勘查蘭嶼之星之狀況，研判是否
                          打開記憶的深鎖，回想起從 98
                                                                  有其他更有效執行方式，無奈最終仍
                     年元月份開始接手承辦該案起，已陸
                                                                  未能拍定，移送機關亦不再申請續拍。
                     續拍出義務人公司名下船舶海晏 12
                     號、占岸五號及占岸八號。查封中之                                船舶基本資料如下：

                     慮長期停泊未使用，恐影響其他船隻                                ( 二 ) 船種：客船
                     進出港，為維護港區安全，港區主管                                ( 三 ) 總噸位：197 噸
                     機關將蘭嶼之星停泊在該漁港最內側                                ( 四 ) 船舶國籍：中華民國
